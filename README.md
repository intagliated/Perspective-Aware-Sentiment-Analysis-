
# Perspective-Aware Sentiment Analysis

##  Project Goal

To demonstrate the critical difference between traditional transformer-based models and Large Language Models (LLMs) in sentiment analysis, specifically focusing on the lack of **perspective awareness** in classic models.

### 1. Traditional Model (Twitter-RoBERTa)

* **Role:** Serves as a strong baseline, highly effective for general, short social media text
* **Limitation:** **Not perspective-aware**
  
### 2. LLM Approach (GPT)

* **Role:** Demonstrates true contextual interpretation through prompt engineering
* **Key Advantage:** It dynamically adjusts sentiment based on the defined reader's identity, capturing domain-specific nuances like sports rivalry, market behavior, and policy effects
